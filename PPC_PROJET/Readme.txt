Simulation de marché énergitique. 
La présentation du projet est dans le pptx du repository. 
Lancement du programme : pyhton3 main.py --> Les indications sont écrites dans le main
Pour simuler un TREMBLEMENT de terre : Le numéro du PID est indiqué dans le main lors du lancement du programme
  --> Signal a envoyer USR1 : kill -s USR1 [NumeroPIDMarket]
Il faut installer au préalable le module matplotlib pour l'affichage graphique.
